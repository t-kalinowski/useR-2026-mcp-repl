# `mcp-repl`: a sandboxed R runtime for LLM agents

Slides from a lightning talk at useR! 2026 in Warsaw.

[View the presentation](https://t-kalinowski.github.io/useR-2026-mcp-repl/)

The presentation introduces [`mcp-repl`](https://github.com/posit-dev/mcp-repl),
an MCP server that gives LLM agents a persistent, sandboxed R or Python
session.

## Render locally

Install [Quarto](https://quarto.org/), then run:

```sh
quarto render mcp-repl-lightning-talk.qmd
```

This creates a standalone `index.html` with the presentation resources
embedded.
